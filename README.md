# 📊 HR Analytics Dashboard (Power BI)

## Project Overview
This project presents an HR analytics dashboard built using Power BI and a structured data model.
The dashboard focuses on analyzing workforce structure, employee attrition, and performance & engagement metrics to support data-driven HR decisions.

---

## Key Features
- Workforce overview (Headcount, Salary, Demographics)
- Attrition analysis (rate, trends, key reasons)
- Performance evaluation (meeting expectations, high/low performers)
- Engagement analysis by department
- Interactive filtering (Department, Year, Age group)

---

## Data Preparation
- Cleaned and transformed employee dataset using Power Query
- Built a star schema data model (fact + dimension tables)
- Created calculated columns for performance categorization
- Developed DAX measures for key HR metrics

---

## Tools
- Power BI
- DAX
- Power Query
- Data Modeling

---

## Data Source
- HR dataset from kaggle

## Key Metrics
- Headcount
- Average Salary
- Attrition Rate
- Average Tenure
- Average Performance Score
- % Meeting Expectations
- % High Performers
- % Low Performers

---

## Insights
- Majority of employees meet performance expectations (~90%)
- High performers represent a small portion of the workforce (~12%)
- Attrition is highest in the Production department
- Engagement levels differ across departments
